# Digital-SamtalsYta
Ett digitalt verktyg som hjälper till att öka delaktighet för våra LSS deltagare. Verktyget är byggd på sättet där LSS personal kan använda bildstöd, kombinerat med specifika frågor för att få mer information från våra deltagare med begränsad språkkunskap, tal mm. Verktyget ska kunna förberedas i förväg och kan delas till individens iPad, telefon..
